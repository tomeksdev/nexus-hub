NexusHub is coming soon

A self-hosted WireGuard management platform with a Go backend, React dashboard, API and CLI workflows, observability, and programmable eBPF security rules. NexusHub is currently under active development.

Features, screenshots, and release packages may change before the first stable version.
Heads up — v2.0.0 is a full rewrite. The v1.0.0 bash installer + Python WebGUI is being replaced. Legacy files remain in git history for reference.
